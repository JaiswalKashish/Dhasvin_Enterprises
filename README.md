## 📌 Project Overview

This system is built to solve real-world inventory challenges faced by businesses. It integrates multiple modules into a single platform, reducing manual work and improving efficiency.

The application supports **Excel and PDF-based data processing**, making it highly practical for real business environments.

---

## ✨ Key Features

### 📦 Inventory Management
- Add / update products
- Real-time stock tracking
- Prevent negative stock

### 🧾 Billing System
- Generate GST invoices
- Auto-update inventory
- Store sales records

### 📥 Purchase Management
- Bulk upload via Excel
- Auto stock updates
- Maintain purchase history

### 👨‍💼 Supplier Management
- Store supplier details
- Centralized vendor database

### 📊 Analytics Dashboard
- Sales & revenue insights
- Product performance tracking
- Visual charts (Recharts)

### 📄 Smart Data Upload
- Upload Excel files
- Upload PDF invoices
- Convert PDF → Excel for structured processing
- Preview before final submission

### 🔐 Role-Based Access
- Admin (full access)
- Staff (limited access)

---

## 🧠 Advanced Features

- Module-based PDF scanning (Products, Sales, Purchase, Supplier)
- Intelligent data parsing
- Real-time system updates
- Dark / Light mode UI
