SMART IMPORT / BILL ANALYSIS WORKFLOW
When a file is uploaded, the system should:
Step 1 – Read uploaded file
Parse uploaded file using:
•	xlsx for Excel 
•	csv-parser for CSV 
•	pdf-parse for PDF text extraction 
Step 2 – Extract data fields
System should identify and extract fields such as:
•	Product Name 
•	Category 
•	HSN/SAC Code 
•	Barcode 
•	Type 
•	Unit Price 
•	Price with Tax 
•	Tax 
•	Qty 
•	Units 
•	Discount 
•	Discount Amount 
•	Purchase Unit Price 
•	Purchase Price With Tax 
•	Description 
•	Not For Sale 
Step 3 – Data Mapping Preview
Show a beautiful “Preview Imported Data” screen where user can:
•	review extracted rows 
•	edit wrong values 
•	map columns if needed 
•	remove invalid rows 
•	validate missing fields 
Step 4 – Smart Inventory Sync
After confirmation, system should:
•	check if product already exists 
•	update stock if existing product is found 
•	insert new product if product is not found 
•	update purchase price if needed 
•	preserve product metadata 
•	create purchase records automatically 
Step 5 – Auto Stock Update
When import/purchase is confirmed:
•	stock quantity should automatically increase 
•	purchase records should be stored 
•	analytics should refresh immediately

6️⃣ SMART INSIGHTS / BUSINESS INTELLIGENCE
Calculate these using actual logic:
Fast Moving Product
Product with highest total quantity sold.
Least Selling Product
Product with lowest quantity sold.
Dead Stock
Product that:
•	has stock available 
•	but has no sales in recent selected period 
Low Stock
If:
quantity < reorder level
Out of Stock
If:
quantity = 0
Restock Suggestion
Suggest restock if:
quantity < reorder level
OR
average recent sales > available stock
Top Category
Category with highest total sales.
Inventory Value
Use:
Sum(quantity × purchase_unit_price)
Revenue
Use:
Sum(quantity_sold × selling_price)
Stock Health Score
Create a score based on:
•	low stock count 
•	dead stock count 
•	out of stock count 
•	total active products
