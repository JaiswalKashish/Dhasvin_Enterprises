SMART IMPORT LOGIC
--------------------------------------------------

When a file is uploaded:

### Step 1 — Parse the file
Read all rows properly and validate them.

### Step 2 — For each row:
Do the following automatically:

#### PRODUCT HANDLING
- If product already exists (based on product name or barcode or SKU if available):
  - update existing product
  - update stock quantity
  - update price if needed
  - update category/supplier mapping if needed
- If product does not exist:
  - create a new product automatically

#### CATEGORY HANDLING
- If category exists:
  - link product to existing category
- If category does not exist:
  - create category automatically

#### SUPPLIER HANDLING
- If supplier column exists:
  - if supplier already exists, link it
  - if supplier does not exist, create it automatically

#### STOCK HANDLING
- If quantity is present:
  - set or update stock quantity
- If reorder level is present:
  - save reorder level
- Calculate stock status automatically

AUTO STOCK STATUS LOGIC
--------------------------------------------------

Determine product stock status automatically based on quantity and reorder level:

1. In Stock

quantity > reorder level

2. Low Stock
